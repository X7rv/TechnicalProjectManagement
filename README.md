# نظام إدارة الطلبات

نظام إدارة طلبات احترافي مكتوب بـ Java Swing. عربي/إنجليزي، RTL/LTR، بدون أي مكتبات خارجية.

## التشغيل
Windows:** `run.bat`
Linux/Mac: `./run.sh`

تحتاج Java 17 أو أحدث.

## الصفحات
نظرة عامة — KPIs + تحليلات + رسوم بيانية + تنبيهات ذكية
الطلبات** — إضافة من المخزون / مخصصة، بحث، فلتر، تعديل، تكرار، حذف متعدد
المنتجات — مخزون مع تنبيهات نفاد/منخفض
العملاء— تلقائي من الطلبات، رتب Bronze/Silver/Gold/Platinum
الإعدادات — 9 خيارات

## الميزات الرئيسية
 تراجع كامل (Ctrl+Z) - يحفظ آخر 50 عملية
 نسخ احتياطية يومية تلقائية في `data/backups/`
ربط الطلبات بالمخزون (إنقاص + استعادة تلقائي)
ضريبة VAT + خصم على الطلب
اختصارات لوحة المفاتيح
ثيم داكن/فاتح 3 أحجام خط

## اختصارات
Ctrl+N` — طلب من المخزون
Ctrl+Shift+N` — طلب مخصص
Ctrl+D` — تكرار
Ctrl+F` — بحث
Ctrl+Z` — تراجع
Delete` — حذف
Double-click — تعديل

## البيانات
data/orders.json` — الطلبات
data/products.json` — المنتجات
data/settings.json` — الإعدادات
data/backups/` — النسخ الاحتياطية اليومية
